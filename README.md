# HelloWorld

Java编程第一步，HelloWorld

在IDEA或者记事本中编辑代码，**HelloWorld.java**

**代码**：[HelloWorldJava.java](./BasicGrammar/Hello/HelloWorldJava.java)

**在命令行执行：**

javac HelloWorld.java //生成 *.class* 文件

java HelloWorld.class //执行Java程序

执行完成。
